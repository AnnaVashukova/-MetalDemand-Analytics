# Документация дашборда "Анализ спроса в металлургической отрасли"

## Назначение дашборда

Дашборд "Металлургия" предназначен для анализа емкости рынка металлургической продукции в России. Сервис позволяет отслеживать динамику спроса по различным товарным позициям металлургической отрасли как в целом по стране, так и в разрезе регионов.

**Основная цель**: представить инструмент для сбора и анализа данных о спросе на металлургическую продукцию с максимальной детализацией по времени и регионам.

## Ключевые возможности

### 1. Анализ структуры спроса по товарам
- Просмотр долевого распределения спроса между основными товарными группами:
  - Концентрат железорудный
  - Прокат
  - Сталь нелегированная
  - Чугун
  - Трубы стальные
  - Сталь легированная
  - Руда железная
  - Сталь нержавеющая

### 2. Региональный анализ
- Идентификация топ-10 регионов по объему спроса
- Сравнительный анализ спроса между регионами

### 3. Временной анализ
- Отслеживание динамики спроса с 2017 по 2024 год

### 4. Детализированная аналитика
- Возможность фильтрации по годам и месяцам
- Сравнение показателей между различными периодами

## Как пользоваться дашбордом

### Фильтрация данных
1. **Выбор временного периода**:
   - Используйте фильтры "Год" и "Месяц" для настройки анализируемого периода
   - Доступны данные с 2017 по 2024 год

2. **Анализ товарных групп**:
   - На главном экране отображается распределение спроса по товарам
   - Используйте интерактивные элементы для выделения конкретных товаров

### Работа с визуализациями

#### 1. Горизонтальная гистограмма "Спрос по товарам"
- **Назначение**: отображает абсолютные объемы спроса по каждой товарной группе
- **Как использовать**:
  - Читайте значения по горизонтальной шкале (от 0 до 700K тыс. тонн)
  - Сравнивайте длину столбцов для оценки объемов между товарами
- **Интерпретация**: более длинный столбец = больший объем спроса

#### 2. Линейный график "Спрос по товарам в динамике"
- **Назначение**: показывает изменение спроса по годам для различных товаров
- **Как использовать**:
  - Наведите курсор на линию в конкретном году для точных значений
  - Сравнивайте траектории линий для анализа трендов
- **Интерпретация**: восходящий тренд = рост спроса, нисходящий = снижение
- 
#### 3. Круговая диаграмма "Доли спроса по товарам"
- **Назначение**: показывает распределение спроса между товарными группами в процентах
- **Как использовать**:
  - Наведите курсор на сегмент диаграммы для просмотра точного значения доли
  - Кликните по сегменту для выделения конкретной товарной группы
- **Интерпретация**: чем больше сегмент, тем выше доля товара в общем спросе

#### 4. Таблица "Топ 10 регионов по спросу"
- **Назначение**: ранжирование регионов по объему спроса
- **Как использовать**:
  - Анализируйте рейтинг регионов от наибольшего к наименьшему
  - Обращайте внимание на абсолютные значения в тыс. тонн
- **Интерпретация**: регионы с наибольшими значениями - ключевые рынки сбыта

#### 5. Сводная таблица "Спрос по региону и дате"
- **Назначение**: детализированные данные по всем регионам за каждый год
- **Как использовать**:
  - Найдите нужный регион в первом столбце
  - Проследите динамику по горизонтали (с 2017 по 2024 год)
  - Сравните показатели между регионами
- **Интерпретация**: позволяет анализировать региональные тренды


## Методология

- **Единицы измерения**: тыс. тонн
- **Периодичность данных**: ежемесячная/годовая
- **Географическое покрытие**: все регионы РФ
- **Временной охват**: 2017-2024 годы
